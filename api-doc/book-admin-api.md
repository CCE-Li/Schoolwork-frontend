# ~~后台图书管理接口文档 (Admin Book Management API)~~

## 基础信息
- **Base URL**: `/api/admin/books`
- 所有接口都需要 `ADMIN` 角色权限

## 接口列表

### 1. 添加新图书
- **URL**: `POST /api/admin/books`
- **描述**: 添加一本新图书
- **请求体**: Book 对象
- **响应**: 添加成功的图书信息

### 2. 更新图书信息
- **URL**: `PUT /api/admin/books/{id}`
- **描述**: 更新指定ID的图书信息
- **路径参数**:
  - `id`: 图书ID
- **请求体**: 更新后的 Book 对象
- **响应**: 更新后的图书信息

### 3. 删除图书
- **URL**: `DELETE /api/admin/books/{id}`
- **描述**: 删除指定ID的图书
- **路径参数**:
  - `id`: 图书ID
- **响应**: 操作结果

### 4. 更新图书状态
- **URL**: `PUT /api/admin/books/{id}/status/{status}`
- **描述**: 更新图书的上下架状态
- **路径参数**:
  - `id`: 图书ID
  - `status`: 状态码 (0:下架, 1:上架)
- **响应**: 操作结果

### 5. 获取图书详情
- **URL**: `GET /api/admin/books/{id}`
- **描述**: 根据ID获取图书详情
- **路径参数**:
  - `id`: 图书ID
- **响应**: 图书详情

### 6. 分页查询图书列表
- **URL**: `GET /api/admin/books`
- **描述**: 分页获取图书列表
- **请求参数**:
  - `pageNum` (可选, 默认值: 1): 页码
  - `pageSize` (可选, 默认值: 10): 每页数量
  - `keyword` (可选): 搜索关键词
- **响应**: 分页的图书列表

### 7. 修改图书价格
- **URL**: `PUT /api/admin/books/{id}/price`
- **描述**: 修改指定ID的图书价格
- **路径参数**:
  - `id`: 图书ID
- **请求参数**:
  - `price`: 新价格
- **响应**: 操作结果

## 公共响应格式
```json
{
  "code": 200,
  "message": "success",
  "data": {} // 具体数据
}
```

## 错误码说明
- 200: 请求成功
- 401: 未授权
- 403: 禁止访问
- 404: 资源不存在
- 500: 服务器内部错误
